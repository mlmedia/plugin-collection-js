Flyout menu plugin demos

Below are demos of the jQuery infinite-scrolling flyout menu plugin from the public GitHub project found here: https://github.com/mlmedia/js-plugin-flyout-menu. Each section displays a different instance of the plugin determined by the initialization options and CSS.

The combined CSS for all of the demos can be viewed here: http://demo.dockstreetmedia.com/js-plugin-flyout-menu/css/js-plugin-flyout-menu.css

NOTE: the CSS should work for all modern browsers (Chrome, Firefox, Safari, etc.) and Internet Explorer 8 and later. IE 7 and older, things fall apart. However, if you are still using IE 7, then I'm sorry, but that's your fault.

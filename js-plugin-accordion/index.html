Accordion plugin demos

Below are demos of the accordion plugin from the public GitHub project found here: https://github.com/mlmedia/accordion. Each section displays a different instance of the plugin determined by the initialization options and CSS.

Accordion DEMO #1

Unstyled accordion with default options: no animation, no multiple box-opening capability.

Accordion DEMO #2

Styled accordion with some basic CSS.

Accordion DEMO #3

Styled accordion with multiple box-opening capability.

Accordion DEMO #4

Styled accordion with multiple box-opening capability and animation.

Accordion DEMO #5

Styled accordion with multiple box-opening capability and animation, using Font Awesome icons in the "expand" and "collapse" elements.
